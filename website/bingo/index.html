Bingo

Host or join a live Bingo game.

To start a game:

  1. Go to Host and login
  2. Grab the Room Code
  3. Select your game options
  4. Have your players go to Join Game
  5. Players will need to enter their username and Room Code
  6. Host press Start Game
    Host can either call number individually or have them auto called
